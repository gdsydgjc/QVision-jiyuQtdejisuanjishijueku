# QVision-基于Qt的计算机视觉库

## 介绍
QVision 是一个基于 QT 的面向对象的多平台计算机视觉库。它旨在为开发者提供一个方便易用的工具，用于创建图形化应用程序。QVision 的算法库主要借鉴了 OpenCV、GSL、CGAL、IPP、Octave 等高性能库，确保了其在计算机视觉任务中的高效性和可靠性。

## 特点
- **多平台支持**：QVision 可以在多个平台上运行，包括 windows、Linux 和 macOS。
- **面向对象设计**：采用面向对象的设计方法，使得代码结构清晰，易于维护和扩展。
- **图形化应用程序**：通过 QT 框架，可以轻松创建具有图形用户界面的应用程序。
- **高性能算法**：借鉴了多个高性能库的算法，确保了在计算机视觉任务中的高效处理能力。

## 适用场景
QVision 适用于需要进行图像处理、计算机视觉任务的应用程序开发。无论是学术研究、工业应用还是个人项目，QVision 都能提供强大的支持。
